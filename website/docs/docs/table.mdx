# Table

Properties related to table container.

### `tableStyle` {#tableStyle}

- Type: [`CSSStyle`](./sharedTypes#CSSStyle)

Object containing CSS styling properties for the table element. <br />

#### Example

import ActiveTable from '@site/src/components/table/activeTableBrowser';
import TableContainer from '@site/src/components/table/tableContainer';
import LineBreak from '@site/src/components/markdown/lineBreak';
import BrowserOnly from '@docusaurus/BrowserOnly';
import TabItem from '@theme/TabItem';
import Tabs from '@theme/Tabs';

<BrowserOnly>{() => require('@site/src/components/nav/autoNavToggle').readdAutoNavShadowToggle()}</BrowserOnly>

<TableContainer>
  <ActiveTable
    tableStyle={{
      borderRadius: '10px',
      boxShadow: 'rgb(172 172 172 / 17%) 0px 0.5px 1px 0px',
      width: '100%',
    }}
    content={[
      ['Planet', 'Diameter', 'Mass', 'Moons', 'Density'],
      ['Earth', 12756, 5.97, 1, 5514],
      ['Mars', 6792, 0.642, 2, 3934],
      ['Jupiter', 142984, 1898, 79, 1326],
      ['Saturn', 120536, 568, 82, 687],
      ['Neptune', 49528, 102, 14, 1638],
    ]}
  ></ActiveTable>
</TableContainer>

### `overflow` {#overflow}

- Type: { `maxWidth?: string`, `maxHeight?: string` }
- Default: _unset_

A scrollbar is introduced on the table if either of the specified dimensions are exceeded. The use of this object automatically sets
[`stickyHeader`](./header#stickyHeader) to _true_ which can be set back to _false_ if required.

      width: '1000px',

    content={[
      ['Planet', 'Diameter', 'Mass', 'Moons', 'Density'],
      ['Earth', 12756, 5.97, 1, 5514],
      ['Mars', 6792, 0.642, 2, 3934],

      ['Neptune', 49528, 102, 14, 1638],

      ['Uranus', 51118, 86.8, 27, 1270],
      ['Pluto', 2376, 0.013, 5, 1850],
      ['Moon', 3475, 0.073, 0, 3340],
    ]}
  ></ActiveTable>
</TableContainer>

### `frameComponentsStyles` {#frameComponentsStyles}

- Type: {<br />
  &nbsp;&nbsp;&nbsp;&nbsp; [`styles?: HoverableStyles`](./sharedTypes#HoverableStyles), <br />
  &nbsp;&nbsp;&nbsp;&nbsp; `inheritHeaderColors?: boolean` <br />
  }
- Default: _{ inheritHeaderColors: true }_

Frame components are the index column, add new column and the add new row. This property is used to set their styling. <br />
The `styles` property defines components' styling for varying mouse states. <br />
`inheritHeaderColors` is used to toggle whether the header cells of the index column and the add new column would inherit the header
row's `backgroundColor` and text `color`.

#### Example

<TableContainer>
  <ActiveTable
    frameComponentsStyles={{
      styles: {
        default: {backgroundColor: '#f5f5f5'},
        hoverColors: {backgroundColor: '#dedede'},
      },
      inheritHeaderColors: false,
    }}
    content={[
      ['Planet', 'Diameter', 'Mass', 'Moons', 'Density'],
      ['Earth', 12756, 5.97, 1, 5514],
      ['Mars', 6792, 0.642, 2, 3934],
      ['Jupiter', 142984, 1898, 79, 1326],
      ['Saturn', 120536, 568, 82, 687],
      ['Neptune', 49528, 102, 14, 1638],
    ]}
    tableStyle={{width: '100%', boxShadow: 'rgb(172 172 172 / 17%) 0px 0.5px 1px 0px', borderRadius: '2px'}}
  ></ActiveTable>
</TableContainer>

### `auxiliaryStyle` {#auxiliaryStyle}

- Type: `string`

This is primarily used to configure [`Webkit CSS`](https://developer.mozilla.org/en-US/docs/Web/CSS/WebKit_Extensions) which cannot be added through component's style properties. A great
example of this is the [`::-webkit-scrollbar`](https://developer.mozilla.org/en-US/docs/Web/CSS/::-webkit-scrollbar) pseudo element which affects the style of the table's scrollbar.

#### Example

<TableContainer>
  <ActiveTable
    overflow={{maxHeight: '300px', maxWidth: '100%'}}
    tableStyle={{
      boxShadow: 'rgb(172 172 172 / 17%) 0px 0.5px 1px 0px',
      width: '1000px',
    }}
    content={[
      ['Planet', 'Diameter', 'Mass', 'Moons', 'Density'],
      ['Earth', 12756, 5.97, 1, 5514],
      ['Mars', 6792, 0.642, 2, 3934],
      ['Jupiter', 142984, 1898, 79, 1326],
      ['Saturn', 120536, 568, 82, 687],
      ['Neptune', 49528, 102, 14, 1638],
      ['Mercury', 4879, 0.33, 0, 5429],
      ['Venus', 12104, 4.87, 0, 5243],
      ['Uranus', 51118, 86.8, 27, 1270],
      ['Pluto', 2376, 0.013, 5, 1850],
      ['Moon', 3475, 0.073, 0, 3340],
    ]}
    auxiliaryStyle="
      ::-webkit-scrollbar {
        width: 10px;
        height: 10px;
      }
      ::-webkit-scrollbar-thumb {
        background-color: #54a7ff;
        border-radius: 5px;
      }"
  ></ActiveTable>
</TableContainer>
