# CSV

You can import/export CSV (Excel) files via the use of internal buttons or component [methods](./methods#importCSV). Additionally, you can
insert CSV data by simply copying cells from Excel and pasting them into the table directly.

### `csvButtons` {#csvButtons}

- Type: {<br />
  &nbsp;&nbsp;&nbsp;&nbsp;`import?`: `boolean` | { [`CSVButton?`](#CSVButton), [`overwriteOptions?: ImportOverwriteOptions`](./sharedTypes#ImportOverwriteOptions) },<br />
  &nbsp;&nbsp;&nbsp;&nbsp;`export?`: `boolean` | { [`CSVButton?`](#CSVButton), `fileName?: string` } <br />
  }

`import` and `export` buttons can be enabled by setting their properties to _true_ or by definining their custom configurations using json objects. Both of these objects
can optionally contain the properties inside the [CSVButton](#CSVButton) type. Additionally, the `import` object can contain the [`overwriteOptions`](./sharedTypes#ImportOverwriteOptions)
property which defines the configuration of what is overwritten and `export` can contain a `fileName` property to define the exported file's name
e.g. _custom_file_name_ or _custom_file_name.csv_ (default is _table_data.csv_).

## Types

Object types for properties inside [`csvButtons`](#csvButtons):

### `CSVButton` {#CSVButton}

- Type: {<br />
  &nbsp;&nbsp;&nbsp;&nbsp;[`styles?: StatefulCSS`](./sharedTypes#StatefulCSS), <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`text?: string`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;[`position?: OuterContentPosition`](./sharedTypes#OuterContentPosition), <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`order?: number` <br />
  }
- Default: {<br />
  &nbsp;&nbsp;&nbsp;&nbsp;_text: "Import CSV"|"Export CSV"_, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;_position: "bottom-left"_, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;_order: 0_ <br />
  }

Custom settings for the `csvButtons` properties. <br />
`styles` is the definition of button style for various mouse events. <br />
`text` is the text that is displayed before the button. <br />
`position` is the location of the button. <br />
`order` is the order of the button when multiple other components are also placed in the same position.
