# Installation

Install the component via [npm](https://www.npmjs.com/):

```bash
npm install active-table
```

For React, install the following instead: <br />

```bash
npm install active-table-react
```

Access the component via CDN:

```
https://unpkg.com/active-table@0.2.5/dist/activeTable.bundle.js
```
