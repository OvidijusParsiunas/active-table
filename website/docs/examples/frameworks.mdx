## Angular

Please add the `CUSTOM_ELEMENTS_SCHEMA` into your module's `schemas` array and import the component
as `import 'active-table';`. See this [blog](https://www.thisdot.co/blog/how-to-integrate-web-components-using-lit-in-angular) for more information.

[Click for Live Example](https://codesandbox.io/s/active-table-angular-9v8nfe?file=/src/app/app.component.html)

## Vanilla JS

The Live Example demonstrates how to set values via properties (recommended) and attributes, which should additionally be used as a working
fallback approach for other frameworks. This example also demonstrates how to import the component via a CDN.

[Click for Live Example](https://codesandbox.io/s/active-table-vanillajs-62yrrj?file=/index.html)

## Troubleshooting

If your platform has not been listed and Vanilla JS approach does not work as fallback, or if you are having issues with any of
the existing examples, please create a [**new issue ticket**](https://github.com/OvidijusParsiunas/active-table/issues/new/choose) on GitHub.
