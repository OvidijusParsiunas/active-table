# Header

Properties related to the header cells.

### `headerStyles` {#headerStyles}

- Type: [`HoverableStyles`](./sharedTypes#HoverableStyles)
- Default: _unset_

Styling details for the header elements in default and mouse hover states. To prevent the frame
component (index and add new column) from inheriting the header style, set `inheritHeaderColors` property
in [`frameComponentsStyle`](./table#frameComponentsStyle) to _false_.

#### Example

import ActiveTable from '@site/src/components/table/activeTableBrowser';
import TableContainer from '@site/src/components/table/tableContainer';
import LineBreak from '@site/src/components/markdown/lineBreak';
import BrowserOnly from '@docusaurus/BrowserOnly';
import TabItem from '@theme/TabItem';
import Tabs from '@theme/Tabs';

<BrowserOnly>{() => require('@site/src/components/nav/autoNavToggle').readdAutoNavToggle()}</BrowserOnly>

<TableContainer>
  <ActiveTable
    headerStyles={{
      default: {
        backgroundColor: '#efefef',
        borderBottom: 'unset',
        fontWeight: '800',
        fontSize: '15px',
      },
      hoverColors: {
        backgroundColor: '#d2d2d2',
      },
    }}
    content={[
      ['Planet', 'Diameter', 'Mass', 'Moons', 'Density'],
      ['Earth', 12756, 5.97, 1, 5514],
      ['Mars', 6792, 0.642, 2, 3934],
      ['Jupiter', 142984, 1898, 79, 1326],
      ['Saturn', 120536, 568, 82, 687],
      ['Neptune', 49528, 102, 14, 1638],
    ]}
    tableStyle={{width: '100%', boxShadow: 'rgb(172 172 172 / 17%) 0px 0.5px 1px 0px', borderRadius: '2px'}}
  ></ActiveTable>
</TableContainer>

### `allowDuplicateHeaders`

- Type: `boolean`
- Default: _true_

Allows multiple columns to have the same header text. If set to _false_ and a duplicate is found the header text is defaulted to the value inside
[`defaultText`](./content#defaultText).

#### Example for false

<TableContainer>
  <ActiveTable
    allowDuplicateHeaders={false}
    content={[
      ['Planet', 'Planet', 'Moons', 'Moons', 'Density'],
      ['Earth', 12756, 5.97, 1, 5514],
      ['Mars', 6792, 0.642, 2, 3934],
      ['Jupiter', 142984, 1898, 79, 1326],
      ['Saturn', 120536, 568, 82, 687],
      ['Neptune', 49528, 102, 14, 1638],
    ]}
    tableStyle={{width: '100%', boxShadow: 'rgb(172 172 172 / 17%) 0px 0.5px 1px 0px', borderRadius: '2px'}}

</TableContainer>

```html

### `headerIconStyle` {#HeaderIconStyle}

- Type: {<br />
  &nbsp;&nbsp;&nbsp;&nbsp; `filterColor?: string`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp; `scale?`: {`x?: number`, `y?: number`}<br />
  }
- Default: {<br />
  &nbsp;&nbsp;&nbsp;&nbsp; _iconFilterColor: 'brightness(0) saturate(100%) invert(34%) sepia(0%) saturate(1075%) hue-rotate(211deg) brightness(96%) contrast(90%)'_, <br />
  &nbsp;&nbsp;&nbsp;&nbsp; _scale: {x: 1.1, y: 1.1}_<br />
  }

Styling details for the column header icons. The color is set by using the `filterColor` property which requires
a [filter](https://developer.mozilla.org/en-US/docs/Web/CSS/filter) string (the reason for this is because the icon is an SVG element)
that can be retrieved using the [cssfilterconverter](https://cssfilterconverter.com/) tool . Use `scale` to resize the icon dimensions.

#### Example

<TableContainer>
  <ActiveTable
    headerIconStyle={{
      filterColor:
        'brightness(0) saturate(100%) invert(16%) sepia(85%) saturate(1892%) hue-rotate(210deg) brightness(88%) contrast(103%)',
      scale: {x: 1.5, y: 1.5},
    }}
    content={[
      ['Planet', 'Diameter', 'Mass', 'Moons', 'Density'],
      ['Earth', 12756, 5.97, 1, 5514],
      ['Mars', 6792, 0.642, 2, 3934],
      ['Jupiter', 142984, 1898, 79, 1326],
      ['Saturn', 120536, 568, 82, 687],
      ['Neptune', 49528, 102, 14, 1638],
    ]}
    tableStyle={{width: '100%', boxShadow: 'rgb(172 172 172 / 17%) 0px 0.5px 1px 0px', borderRadius: '2px'}}
  ></ActiveTable>
</TableContainer>

<Tabs>
<TabItem value="js" label="Sample code">

  }'
></active-table>
```

### `stickyHeader` {#stickyHeader}

- Type: `boolean`
- Default: _false_ or if [`overflow`](./table#overflow) property is defined set to _true_

Header row will remain at the top of the table when there is overflow in the table or a parent element.

#### Example

<TableContainer customStyle={{maxHeight: '200px', overflow: 'auto'}}>
  <ActiveTable
    stickyHeader={true}
    content={[
      ['Planet', 'Diameter', 'Mass', 'Moons', 'Density'],
      ['Earth', 12756, 5.97, 1, 5514],
      ['Mars', 6792, 0.642, 2, 3934],
      ['Jupiter', 142984, 1898, 79, 1326],
      ['Saturn', 120536, 568, 82, 687],
      ['Neptune', 49528, 102, 14, 1638],
    ]}
    tableStyle={{width: '97%', boxShadow: 'rgb(172 172 172 / 17%) 0px 0.5px 1px 0px', borderRadius: '2px'}}
  ></ActiveTable>
</TableContainer>

<Tabs>
<TabItem value="js" label="Sample code">

```html
<active-table stickyHeader="true"></active-table>
```

</TabItem>
<TabItem value="py" label="Full code">

```html
<!-- This example is for Vanilla JS and should be tailored to your framework (see Examples) -->

<div style="maxHeight: 200px; overflow: auto">
  <active-table
    stickyHeader="true"
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],
      ["Earth", 12756, 5.97, 1, 5514],
      ["Mars", 6792, 0.642, 2, 3934],
      ["Jupiter", 142984, 1898, 79, 1326],
      ["Saturn", 120536, 568, 82, 687],
      ["Neptune", 49528, 102, 14, 1638]]'
    tableStyle='{"borderRadius":"2px"}'
  ></active-table>
</div>
```

</TabItem>
</Tabs>

:::note
The scrollbar in this example is located on the parent div element, however when using the [`overflow`](./table#overflow) property
it would be placed directly on the actual table component.
:::

### `dataStartsAtHeader` {#dataStartsAtHeader}

- Type: `boolean`
- Default: _false_

Starts the index column count at header row and if [`pagination`](./pagination) is used the header row will not be displayed in pages after the first.
