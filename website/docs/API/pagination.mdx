# Pagination

Divide table rows into a series of pages.

### `pagination` {#pagination}

- Type: {<br />
  &nbsp;&nbsp;&nbsp;&nbsp;`rowsPerPage: number`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`rowsPerPageSelect: boolean|RowsPerPageSelect`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`maxNumberOfVisiblePageButtons: number`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`displayPrevNext: boolean`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`displayFirstLast: boolean`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`displayNumberOfVisibleRows: boolean`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`style: PaginationStyle<StatefulCSSS>`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`positions: PaginationPositions` <br />
  }
- Default: {<br />
  &nbsp;&nbsp;&nbsp;&nbsp;_rowsPerPage: 10_, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;_rowsPerPageSelect: true_, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;_maxNumberOfVisiblePageButtons: 8_, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;_displayPrevNext: true_, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;_displayFirstLast: true_, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;_displayNumberOfVisibleRows: true_, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;_positions: refer to the PaginationPositions type_ <br />
  }

`rowsPerPage` is the number of maximum data rows that can be displayed in a single page. <br />
`rowsPerPageSelect` controls a dropdown that allows the user to change `rowsPerPage`. <br />
`maxNumberOfVisiblePageButtons` is the maximum number of page (number) buttons that are visible, the rest will be hidden. <br />
`displayPrevNext` defines whether the previous and next page buttons are displayed. <br />
`displayFirstLast` defines whether the first and last page buttons are displayed. <br />
`displayNumberOfVisibleRows` displays information on the index of currently visible rows out of the total number of rows in the table. E.g. _1-5 of 20_. <br />
`style` defines the styling properties of pagination components. <br />
`positions` defines pagination components positions. <br />
