# Header

Properties related to the header row.

### `stickyHeader`

- Type: `boolean`
- Default value: _false_ or if WORK `overflow` property is defined set to _true_

Header row will remain at the top of the table when there is overflow in the table or a parent element.

#### Example

import TableContainer from '@site/src/components/table/table-container';
import LineBreak from '@site/src/components/markdown/line-break';
import {ActiveTable} from 'active-table';
import TabItem from '@theme/TabItem';
import Tabs from '@theme/Tabs';

<TableContainer customStyle={{maxHeight: '200px', overflow: 'auto'}}>
  <active-table
    stickyHeader="true"
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],
      ["Earth", 12756, 5.97, 1, 5514],
      ["Mars", 6792, 0.642, 2, 3934],
      ["Jupiter", 142984, 1898, 79, 1326],
      ["Saturn", 120536, 568, 82, 687],
      ["Neptune", 49528, 102, 14, 1638]]'
    tableStyle='{"width":"97%", "backgroundColor": "white", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'
  ></active-table>
</TableContainer>

<Tabs>

```text
<active-table stickyHeader="true"></active-table>
```

</TabItem>

### `dataStartsAtHeader`

- Type: `boolean`
- Default value: _false_

Starts the WORK index column count at header row and if WORK pagination is used the header row is no longer displayed in pages after the first.

#### Example

  <active-table
    dataStartsAtHeader="true"
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],
      ["Earth", 12756, 5.97, 1, 5514],
      ["Mars", 6792, 0.642, 2, 3934],
      ["Jupiter", 142984, 1898, 79, 1326],
      ["Saturn", 120536, 568, 82, 687],
      ["Neptune", 49528, 102, 14, 1638]]'
    tableStyle='{"width":"100%", "backgroundColor": "white", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'

</TableContainer>

### `duplicateHeadersAllowed`

- Type: `boolean`
- Default value: _true_

Allows multiple columns to have the same header text.

#### Example for false
