# Rows

Properties related to table rows.

### `rowHoverStyle`

- Type: `{style: CSSStyle, header: boolean, addNewRowButton: boolean}`
- Default value: _unset_, but if _style_ is set then _header_ and _addNewRowButton_ are _true_

Custom styling for rows when they are hovered with a mouse. `header` property denotes if `style` can be applied to the header row
and `addNewRowButton` denotes if `style` can be applied to the WORK add new row button if it is displayed.

#### Example

import TableContainer from '@site/src/components/table/table-container';
import LineBreak from '@site/src/components/markdown/line-break';
import {ActiveTable} from 'active-table';
import TabItem from '@theme/TabItem';
import Tabs from '@theme/Tabs';

<TableContainer>
  <active-table
    rowHoverStyle='{"style":{"backgroundColor": "#d6d6d630"} }'
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],
      ["Earth", 12756, 5.97, 1, 5514],
      ["Mars", 6792, 0.642, 2, 3934],
      ["Jupiter", 142984, 1898, 79, 1326],
      ["Saturn", 120536, 568, 82, 687],
      ["Neptune", 49528, 102, 14, 1638]]'
    tableStyle='{"width":"100%", "backgroundColor": "white", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'
  ></active-table>
</TableContainer>

### `stripedRows`

- Type: `boolean|{odd: CSSStyle, even, CSSStyle}`
- Default value: _false_, but if set to _true_ then _{even: {backgroundColor: ''}, odd: {backgroundColor: '#dcdcdc7a'}}_

Stripe based styling for rows.

#### Example

<TableContainer>
  <active-table
    stripedRows='{"odd": {"backgroundColor": ""}, "even": {"backgroundColor": "#dcdcdc7a"}}'
    defaultColumnsSettings='{"cellStyle": {"borderBottom": "unset"}}'
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],
      ["Earth", 12756, 5.97, 1, 5514],
      ["Mars", 6792, 0.642, 2, 3934],
      ["Jupiter", 142984, 1898, 79, 1326],
      ["Saturn", 120536, 568, 82, 687],
      ["Neptune", 49528, 102, 14, 1638]]'
    tableStyle='{"width":"100%", "backgroundColor": "white", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'
  ></active-table>
</TableContainer>
