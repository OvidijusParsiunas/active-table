# How To

Values can be passed down into the component using properties or attributes.

### Properties

These are set via data-binding using the syntax of your framework of choice or through a reference to the element.

```
// Data binding (React)
<active-table isCellTextEditable={true}></active-table>

// Element reference
const elementRef = document.getElementById('custom-id');
elementRef.isCellTextEditable = true;
```

:::note
Passing values via an element reference should be done as soon as the element is rendered on the DOM, otherwise changes will not take effect.
:::

### Attributes

Set as a _string_ either directly on the element or via a `setAttribute` method on its reference. When passing JSON objects
make sure that they are first [correctly](https://jsonlint.com/) stringified, functions assigned to properties must not have external references
and all regex values are properly escaped.

```
// Direct
<active-table rowHoverStyle='{"style":{"backgroundColor": "#d6d6d630"}}'></active-table>

// Element reference
const elementRef = document.getElementById('custom-id');
elementRef.setAttribute('rowHoverStyle', '{"style":{"backgroundColor": "#d6d6d630"}}');
```

:::note
Passing values via an element reference should be done as soon as the element is rendered on the DOM, otherwise changes will not take effect.
:::
