# Events

Properties accepting functions that get triggered on event occurrence.

:::info
When using these properties take care not to re-render the table component when updating parent state.
:::

### `onCellUpdate` {#onCellUpdate}

- Type: (`cellUpdate:` { <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`text: string`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`rowIndex: number`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`columnIndex: number`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`updateType: "add"|"update"|"remove"` <br />
  }) => `void`

Triggered when any content within the table has changed. Both `rowIndex` and `columnIndex` begin at _0_
where it is the header row for `rowIndex` and the left-most column for `columnIndex`.

#### Example

import TableContainerEvents from '@site/src/components/table/table-container-events';
import LineBreak from '@site/src/components/markdown/line-break';
import {readdAutoNavToggle} from '@site/src/nav/autoNavToggle';
import {ActiveTable} from 'active-table';
import TabItem from '@theme/TabItem';
import Tabs from '@theme/Tabs';
readdAutoNavToggle();

<TableContainerEvents propertyname={"onCellUpdate"}>
  <active-table
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],
      ["Earth", 12756, 5.97, 1, 5514],
      ["Mars", 6792, 0.642, 2, 3934],
      ["Jupiter", 142984, 1898, 79, 1326],
      ["Saturn", 120536, 568, 82, 687],
      ["Neptune", 49528, 102, 14, 1638]]'
    tableStyle='{"width":"100%", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'
></active-table>

</TableContainerEvents>

<Tabs>
<TabItem value="js" label="Sample">

```text
tableReference.setAttribute('onCellUpdate', (cellUpdate) => { console.log(cellUpdate); });
```

</TabItem>
<TabItem value="py" label="Full Code">

```text
tableReference.setAttribute('onCellUpdate', (cellUpdate) => { console.log(cellUpdate); });
```

</TabItem>
</Tabs>

### `onContentUpdate` {#onContentUpdate}

- Type: (`tableUpdate: (string|number)[][]`) => `void`

Similarly to [`onCellUpdate`](#onCellUpdate) this is triggered when any cell is updated - however this property returns the full table contents 2D array instead.

#### Example

  <active-table
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],
      ["Earth", 12756, 5.97, 1, 5514],
      ["Mars", 6792, 0.642, 2, 3934],

### `onColumnsUpdate` {#onColumnsUpdate}

- Type: (`columnsUpdate`: [`ColumnsDetails`](./types#ColumnsDetails)) => `void`

Triggered when column details are updated. This is particularly useful if the user has made changes to columns and you want to recreate them
in the next session on the initial load.

#### Example

    content='[
