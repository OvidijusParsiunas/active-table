# Events

Properties that accept functions that get triggered when an event occurs.

:::info
When using any of these take care not to re-render the table component if updating state.
:::

### `onCellUpdate` {#onCellUpdate}

- Type: (`cellUpdate:` { <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`text: CellText`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`rowIndex: number`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`columnIndex: number`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp;`updateType: "add"|"update"|"remove"` <br />
  }) => `void`

Triggered when any content within the table has changed.

#### Example

import TableContainer from '@site/src/components/table/table-container';
import TableContainerEvents from '@site/src/components/table/table-container-events';
import LineBreak from '@site/src/components/markdown/line-break';
import {ActiveTable} from 'active-table';
import TabItem from '@theme/TabItem';
import Tabs from '@theme/Tabs';

<TableContainerEvents>
  <active-table
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],
      ["Earth", 12756, 5.97, 1, 5514],
      ["Mars", 6792, 0.642, 2, 3934],
      ["Jupiter", 142984, 1898, 79, 1326],
      ["Saturn", 120536, 568, 82, 687],
      ["Neptune", 49528, 102, 14, 1638]]'
    tableStyle='{"width":"100%", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'
></active-table>

</TableContainerEvents>
