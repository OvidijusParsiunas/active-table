# Content

Properties related to text displayed in the table.

### `content`

- Type: `(string|number)[][]`
<!-- TableContent -->

Header and data text to be displayed in the table. First array represents the header row, following arrays represent data rows.

#### Example

import TableContainer from '@site/src/components/table/table-container';
import LineBreak from '@site/src/components/markdown/line-break';
import {ActiveTable} from 'active-table';
import TabItem from '@theme/TabItem';
import Tabs from '@theme/Tabs';

<TableContainer>
  <active-table
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],
      ["Earth", 12756, 5.97, 1, 5514],
      ["Mars", 6792, 0.642, 2, 3934],
      ["Jupiter", 142984, 1898, 79, 1326],
      ["Saturn", 120536, 568, 82, 687],
      ["Neptune", 49528, 102, 14, 1638]]'
    tableStyle='{"width":"100%", "backgroundColor": "white", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'
  ></active-table>
</TableContainer>

### `updateCell`

- Type: `{newText: string|number, rowIndex: number, columnIndex: number}`

Update cell text programmatically after the table has been initially rendered without triggering another re-render.

#### Example

<TableContainerDynamicUpdates>
  <active-table
    content='[
      ["Name", "CPU", "Memory", "Disk", "Network"],
      ["Chrome", "4.5%", "1400MB", "0.2MB/s", "1.2Mbps"],
      ["Firefox", "2.5%", "800MB", "0.1MB/s", "0.5Mbps"],
      ["Edge", "5.5%", "1000MB", "1.4MB/s", "0.7Mbps"],
      ["Safari", "1.5%", "1200MB", "1.2MB/s", "0.2Mbps"],
      ["Opera", "3.5%", "400MB", "0.8MB/s", "0.5Mbps"]]'
    tableStyle='{"width":"100%", "backgroundColor": "white", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'
  ></active-table>
</TableContainerDynamicUpdates>

<Tabs>
<TabItem value="js" label="Sample">

```text
<script>
  ...
  activeTableElementRef.setAttribute('updateCell', JSON.stringify({'newText', 1, 1}));
</script>
```

</TabItem>
<TabItem value="py" label="Full code">

```text
<active-table id="active-table"
  content='[
    ["Name", "CPU", "Memory", "Disk", "Network"],
    ["Chrome", "4.5%", "1400MB", "0.2MB/s", "1.2Mbps"],
    ["Firefox", "2.5%", "800MB", "0.1MB/s", "0.5Mbps"],
    ["Edge", "5.5%", "1000MB", "1.4MB/s", "0.7Mbps"],
    ["Safari", "1.5%", "1200MB", "1.2MB/s", "0.2Mbps"],
    ["Opera", "3.5%", "400MB", "0.8MB/s", "0.5Mbps"]]'
  tableStyle='{"borderRadius":"2px"}'
></active-table>

<script>
  function updateCell(tableElement) {
    setTimeout(() => {
      const rowIndex = Math.floor(Math.random() * 5 + 1);
      const columnIndex = Math.floor(Math.random() * 5 + 1);
      let newText = '';
      if (columnIndex === 1) {
        newText = `${Math.round(Math.random() * 20 * 10) / 10}%`;
      } else if (columnIndex === 2) {
        newText = `${Math.round(Math.random() * 1500 * 10) / 10}MB`;
      } else if (columnIndex === 3) {
        newText = `${Math.round(Math.random() * 1.5 * 10) / 10}MB/s`;
      } else {
        newText = `${Math.round(Math.random() * 1.5 * 10) / 10}Mbps`;
      }
      tableElement.setAttribute('updateCell', JSON.stringify({newText, rowIndex, columnIndex}));
      updateCell(tableElement);
    }, 150);
  }
  const activeTableElementRef = document.getElementById('active-table');
  updateCell(activeTableElementRef);
</script>
```

</TabItem>
</Tabs>

:::info
This property should be set as an attribute via JavaScript - see WORK.
:::

### `spellCheck`

- Type: `boolean`
- Default value: _false_

Checks table text for spelling mistakes (based on the browser's language settings) and if found displays a curly red line under the errors.

#### Example

<TableContainer>
  <active-table
    spellCheck="true"
    content='[
      ["Planetasdasd", "Diamasdasdeter", "Masssdfsdf", "Mooffns", "Denssdsdfdsfity"],
      ["asdasdasd", 12756, 5.97, 1, 5514],
      ["Msadasdars", 6792, 0.642, 2, 3934],
      ["Jupitar", 142984, 1898, 79, 1326],
      ["asudhasd", 120536, 568, 82, 687],
      ["saiuhfjduidshfuis", 49528, 102, 14, 1638]]'
    tableStyle='{"width":"100%", "backgroundColor": "white", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'
  ></active-table>
</TableContainer>

<Tabs>
