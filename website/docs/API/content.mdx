# Content

Properties related to content cells displayed in the table.

### `content`

- Type: `(string|number)[][]`
<!-- TableContent -->

Header and data text to be displayed in the table. First array represents the header row, following arrays represent data rows.

#### Example

import LineBreak from '@site/src/components/markdown/lineBreak';
import {readdAutoNavToggle} from '@site/src/nav/autoNavToggle';
import {ActiveTable} from 'active-table';
import TabItem from '@theme/TabItem';
import Tabs from '@theme/Tabs';
readdAutoNavToggle();

<TableContainer>
  <active-table
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],
      ["Earth", 12756, 5.97, 1, 5514],
      ["Mars", 6792, 0.642, 2, 3934],
      ["Jupiter", 142984, 1898, 79, 1326],
      ["Saturn", 120536, 568, 82, 687],
      ["Neptune", 49528, 102, 14, 1638]]'
    tableStyle='{"width":"100%", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'
  ></active-table>
</TableContainer>

### `defaultText` {#defaultText}

- Type: `string|number`
- Default: _""_

Replacement text that is displayed when cell has no text or its type validation has failed. <br />

#### Example

    defaultText="-"
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],

      ["", "", 1898, 79, 1326],
      ["", "", 568, 82, 687],

    tableStyle='{"width":"100%", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'

### `cellStyle` {#cellStyle}

- Type: { `CSSStyle` & `!minWidth` & `!maxWidth` & `!height` & `!minHeight` & `!maxHeight` }

Defines the default style for table cells. <br />
This object contais all `CSSStyle` properties except `minWidth`, `maxWidth`, `height`, `minHeight` and `maxHeight`. <br />
The `width` property supports _px_ and _%_ measurements, however when using _%_ - the [`tableStyle`](./table#tableStyle) object should have its `width` property
also set as otherwise the cells will not be able to calculate a relative width. Additionally, when [`cellStyle`](#cellStyle) - `width` and
[`tableStyle`](./table#tableStyle) - `width` are defined and [`isColumnResizable`](./column#isColumnResizable) is set to _true_, the [`cellStyle`](#cellStyle) - `width`
will be overriden to reach the set table `width`, hence to prevent this make sure to set [`isColumnResizable`](./column#isColumnResizable) to _false_.

#### Example

<TableContainer>
  <active-table
    cellStyle='{"borderBottom": "1px solid black", "borderRight": "1px solid black", "backgroundColor": "#fafafa"}'
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],
      ["Earth", 12756, 5.97, 1, 5514],
      ["Mars", 6792, 0.642, 2, 3934],
      ["Jupiter", 142984, 1898, 79, 1326],
      ["Saturn", 120536, 568, 82, 687],
      ["Neptune", 49528, 102, 14, 1638]]'
    tableStyle='{"width":"100%", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'
  ></active-table>
</TableContainer>

### `isCellTextEditable` {#isCellTextEditable}

- Type: `boolean`
- Default: _true_

Controls if cell text can be edited for both data and header cells. This property is superseded by [`isHeaderTextEditable`](./header#isHeaderTextEditable) for header cells.

#### Example

<TableContainer>
  <active-table
    isCellTextEditable="false"
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],
      ["Earth", 12756, 5.97, 1, 5514],
      ["Mars", 6792, 0.642, 2, 3934],
      ["Jupiter", 142984, 1898, 79, 1326],
      ["Saturn", 120536, 568, 82, 687],
      ["Neptune", 49528, 102, 14, 1638]]'
    tableStyle='{"width":"100%", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'
  ></active-table>
</TableContainer>

### `spellCheck`

- Type: `boolean`
- Default: _false_

Checks table text for spelling mistakes (based on the browser's language settings) and if found displays a curly red line under the errors.

#### Example

<TableContainer>
  <active-table
    spellCheck="true"
    content='[
      ["Planetasdasd", "Diamasdasdeter", "Masssdfsdf", "Mooffns", "Denssdsdfdsfity"],
      ["asdasdasd", 12756, 5.97, 1, 5514],
      ["Msadasdars", 6792, 0.642, 2, 3934],
      ["Jupitar", 142984, 1898, 79, 1326],
      ["asudhasd", 120536, 568, 82, 687],
      ["saiuhfjduidshfuis", 49528, 102, 14, 1638]]'
    tableStyle='{"width":"100%", "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px", "borderRadius":"2px"}'
  ></active-table>
</TableContainer>

<Tabs>

</TabItem>
