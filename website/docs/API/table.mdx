# Table

Properties related to the table:

### `tableStyle` {#tableStyle}

- Type: [`CSSStyle`](./types#CSSStyle)
<!-- CSSObject - min width not allowed -->

Object containing CSS styling properties for the table element. <br />

#### Example

import TableContainer from '@site/src/components/table/table-container';
import {ActiveTable} from 'active-table';
import TabItem from '@theme/TabItem';
import Tabs from '@theme/Tabs';

<TableContainer>
  <active-table
    tableStyle='{
      "borderRadius": "10px",
      "boxShadow": "rgb(172 172 172 / 17%) 0px 0.5px 1px 0px",
      "width":"100%"
    }'
    content='[
      ["Planet", "Diameter", "Mass", "Moons", "Density"],
      ["Earth", 12756, 5.97, 1, 5514],
      ["Mars", 6792, 0.642, 2, 3934],
      ["Jupiter", 142984, 1898, 79, 1326],
      ["Saturn", 120536, 568, 82, 687],
      ["Neptune", 49528, 102, 14, 1638]]'
  ></active-table>
</TableContainer>

:::tip
When table `width` needs to _inherit_ the parent element width, set `width` to 100% as displayed in the example.
:::
:::note
`minWidth`, `height`, `minHeight`, `maxHeight` properties are not supported.
:::
