## React

When using this framework please add the `CUSTOM_ELEMENTS_SCHEMA` into module `schemas` array and import the component
as `import 'active-table';`. Read this [blog](https://www.thisdot.co/blog/how-to-integrate-web-components-using-lit-in-angular) for reference.

[Click for Live Example](https://codesandbox.io/s/interesting-joana-9v8nfe?file=/src/app/app.component.ts)

## Vanilla JS

The live example demonstrates how to set values via both properties (recommended) and attributes and how to import the component via a CDN.
Please note that if you choose to pass values into the component via an element reference, this process should be carried out as soon as
the element is rendered on the DOM, otherwise your changes will not take effect.

[Click for Live Example](https://codesandbox.io/s/active-table-vanillajs-62yrrj?file=/index.html)
